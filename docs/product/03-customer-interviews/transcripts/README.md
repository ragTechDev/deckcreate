# Transcripts

Raw or lightly-cleaned interview transcripts go here, one file per interview (e.g. `01-jane-doe-podcaster.md`).

**Before committing:** redact names/companies if there's any sensitivity, or keep this folder's contents out of git entirely and store transcripts in a shared drive instead — the repo's `.gitignore` already excludes everything in this folder except this README so that's the default. Remove the ignore rule for this path if you decide raw transcripts are fine to commit as-is.

Suggested per-file format:

```markdown
# Interview 01 — [role/segment, not necessarily full name if sensitive]

**Date:**
**Segment:** (which Lean Canvas customer segment)

## Notes/Transcript

...

## Immediate takeaways (write this within an hour of the call)

-
```
